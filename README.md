# Avlo Admin Panel

This is the frontend of the **Avlo** company's automation system. Built with **React**, **Axios**, and designed with modern UI principles, this admin panel is aimed at simplifying the management and automation of company operations.

## Technologies Used

- **React**: For building dynamic and responsive user interfaces.
- **Axios**: For handling HTTP requests and interacting with APIs.
- **HTML & CSS**: For styling the application.
- **JavaScript**: For frontend logic and interactivity.

## Features

- **User Management**: View, add, edit, and delete users.
- **Dashboard**: Display key metrics and statistics in real-time.
- **Reports**: Generate and export reports.
- **Notifications**: Receive important updates through notifications.
- **Authentication**: Secure login functionality for admins.

## Installation

To set up this project locally, follow these steps:

1. Clone the repository:
git clone https://github.com/Otelbayev/avlo.git

Navigate to the project directory:

cd avlo
Install dependencies:

npm install
Run the development server:

npm start
This will open the application in your browser at http://localhost:3000.
